    Agent

    Base Agent class for all conversational participants in coshop.

    Provides conversation-history tracking, checkpoint save/restore, and a hook system (pre/post-generation) shared by both the shopping-assistant policy and the user simulator.

    Agent(verbosity=0, pre_conversation_hooks=[], pre_generation_hooks=[], post_generation_hooks=DEFAULT_POST_GENERATION_HOOKS, checkpoint_file=None, **kwargs)

    Base class for agents that participate in conversations. Provides common conversation tracking functionality.

    load_checkpoint(checkpoint_file=None, turn_idx=None, connection=None)

    Load the agent state from a checkpoint file. Critically, this does not save configs for the agent, so the agent must be initialized with the same configs as the checkpoint.

    Relies on the hook history to load the state by turn.

    Parameters:

    Name Type Description Default
    checkpoint_file Optional[str]

    Path to checkpoint file. If None, uses self.checkpoint_file.

    		 None
    turn_idx Optional[int]

    If provided, only load the state up to this turn. If None, load the entire checkpoint.

    		 None
